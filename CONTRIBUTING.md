# Contributing to adage

Thanks for being part of this project.

This repository is shared by everyone in the course, so the goal is to keep things simple and organized.

To contribute:

1. Fork this repository to your own GitHub account.
2. Create a new branch for your work.
3. Only work inside the folder related to your section.
4. Do not delete or modify other people’s work.
5. Add a short explanation of what you built when you open your pull request.

Your code does not need to be perfect. The goal is to practice working in a shared repository and explaining your work.

If you’re unsure where something belongs, ask before pushing changes.
